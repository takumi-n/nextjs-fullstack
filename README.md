## このリポジトリについて

[fullstack-nextjs-ecommerce](https://github.com/olafsulich/fullstack-nextjs-ecommerce) を簡略化・カスタマイズした Next.js ベースの EC アプリケーションです

## 利用ライブラリ 🔧

このリポジトリで遊んでいるライブラリです

- Next.js(React)
- TypeScript
- Prisma
- NextAuth
- Tailwind
- React Query
- Sentry
- Yup

## 環境構築 💾

1. `.env.example` ファイルをもとに空欄を埋めた `.env` ファイルをディレクトリルートに配置する
2. `make deps`
3. `make generate`
4. `make up`
5. `make dev`
